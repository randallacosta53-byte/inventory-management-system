# Inventory Management System (Python CLI)

## Description
Console-based application to manage inventory, allowing users to add, update, and track products.

## Features
- Add new products
- Update existing stock
- Remove product quantities
- View inventory status
- Modify product details

## Technologies
- Python

## How to Run
1. Clone the repository
2. Run the main file:
```bash
python main.py
```
---
## Purpose

This project was built to practice data handling, process logic, and basic system design.
